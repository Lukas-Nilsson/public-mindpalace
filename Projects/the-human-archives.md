# The Human Archives

Clothing that links to digital stories about science, culture, and invention.

---

## Overview

The product pairs physical hoodies with NFC/QR that open artifact pages (short, factual stories). New artifacts are released in small batches.

---

## My Role

- Product direction and roadmap
- Design system for artifacts and pages
- Frontend engineering and performance
- Content pipeline from markdown → web

---

## What’s Built

- NFC/QR flow → artifact page
- Content model for artifacts (title, summary, sources, media)
- Static site pipeline for reliability and speed
- Early visuals and copy drafts

---

## Stack

- Web: Next.js / Astro (static output)
- Content: Markdown + frontmatter from public repo
- Deploy: GitHub Pages / Actions

---

## Status

- Building public content pipeline
- Prototyping artifact pages and structure
- Preparing first small release
